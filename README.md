## Docker-Compose [ React + Express + MySQL ]

### Requirement
- Docker and Docker-compose

## Initaite and Run
- Clone the project
- **Launch** the app by change folder to project folder and type
	- > Start App `docker-compose up`

## Ports
- Client: 3000
- Server: 5000
- MySQL: 8080
